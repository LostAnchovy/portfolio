Project Description

The Lost Anchovy App is an IOS application that provides locations, recipes, navigation, and tutorials for the Northern California fishing enthusiast, intermediate or beginner.

Building the Wire Frame

Our team uses Sketch to build out the wire frame for the application. The wireFrame provides us with the Visual guidelines that help us code the front, backend and logic of the application.

Coding the Project

The team used Github for version control and communicated through email, chat and designated checkin dates to follow up with the project . Each team member was assigned a different application functions to code out. When the changes where approved it was merged into the master branch.
