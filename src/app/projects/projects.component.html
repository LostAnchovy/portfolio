SF Carnival

Carnaval SF is a web based application that utilizes Angular 5 front end frame work and a Ruby on Rails back end. The Project was composed of 5 developers, a SCRUM master, and a Project Owner working using the SCRUM project framework. The team used multiple technologies such as Mapbox, Angular, ruby on rails, and bootstrap to make the app user friendly, responsive and accessible across many platforms.

The Lost Anchovoy (IOS APP)

Are you a fishing enthusiasts, a beginner, or just a dad that wants to take his kid fishing for the day? The Lost Anchovy is an application to help the average person and the experience fisherman learn learn how to catch, prepare and cook their own catch!

The Lost Anchovy App is an IOS application that provides locations, recipes, navigation, and tutorials for the Northern California fishing enthusiast or beginner. It was built using UIKit, coreLocation, and Mapkit
