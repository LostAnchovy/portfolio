Project Description

Carnival San Francisco is a festival located in the heart of the mission district that celebrates diverse Latin American and Caribbean roots found in the San Francisco Bay Area. The two day festival is composed of dance, music, visual arts, community learning and a parade. The carnival has deep history in San Francisco and is celebrating their 40th anniversary!

Technologies Used

Mapbox is a location data platform for mobile and web applications. The platform provides building blocks to add location features like maps, search, and navigation into any application. The team used the API to layer points in interest inside the boundaries of the carnaval and help attendees navigate.

Angular 5 is a front end Framework used to build dynamic web applications. Angular was the primary front end applications framework used to generate and build out the mobile application, along a ruby on rails backend that was used for administrativ access.
